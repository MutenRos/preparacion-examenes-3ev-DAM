Python es un o de los idiomas de programacion mas versatiles y faciles de aprender, por lo que es extensamente utilizado para una gran variedad de tareas distintas. En el proyecto de hoy, crearemos el programa mas clasico y sencillo con python, un `HOLA_MUNDO`

Se asume Python instalado (por ejemplo Python 3.x). En Raspberry Pi OS suele estar instalado.
De no estarlo, en la consola de nuestra raspberry pi:

```bash
sudo apt install python3-pip
```
Y creamos nuestro script con 

```bash
sudo nano hola_mundo.py
```

2. Código fuente (archivo: `hola_mundo.py`)

```python
print("Hola mundo")
```

Para ejecutarlo, lo guardaremos con Ctrl+O, Enter, Ctrl+X, para salir y sin salirnos de la carpeta escribiremos en el terminal:

```bash
python3 hola_mundo.py
```
Salida esperada en consola:

```
Hola mundo
```

Aunque simple, este puede ser un ejemplo de codigo fuente, interpretado, sin compilar.
