Servidor Flask con acceso IMAP, filtrado IA y web de gestión de correos

📋 Descripción

Aplicación web Python/Flask que conecta a un servidor de correo IMAP, descarga mensajes y los categoriza automáticamente usando un modelo de lenguaje local (Ollama).


Permite al usuario navegar sus correos, responder y gestionar categorías desde una interfaz web.

🛠️ Tecnologías

Python 3 Flask IMAP imaplib Ollama HTML/CSS

🚀 Cómo ejecutar

python -m venv .venv source .venv/bin/activate pip install flask # Configura tus credenciales en config.py python app.py

✨ Características

  • Conexión IMAP segura (SSL)
  • Clasificación automática con IA
  • Vista de bandeja de entrada
  • Redacción y envío de respuestas
  • Filtros por categoría
