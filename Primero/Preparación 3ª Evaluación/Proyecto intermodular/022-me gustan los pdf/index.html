🧰 Toolkit PDF

Caja de herramientas web con 8 utilidades para manipular PDFs: combinar, extraer, comprimir, OCR, firmar y más. Frontend PHP, backend Python con pypdf y Pillow.

📄

8 herramientas, una sola interfaz

Cada acción tiene su propio script Python y su endpoint PHP. Todo desde el navegador, sin instalar nada.
