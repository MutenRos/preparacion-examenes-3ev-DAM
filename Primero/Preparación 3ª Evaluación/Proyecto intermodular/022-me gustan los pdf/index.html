📚 Me gustan los PDF

Toolkit PDF completo: PHP+Python, 8 operaciones (split, merge, convert, resize) con UI de arrastrar y soltar

📖 Qué hace realmente

Una aplicación web donde PHP sirve la interfaz y despacha operaciones a scripts Python. Tiene 8 operaciones reales: dividir PDF, unir PDFs, PDF a imágenes, imágenes a PDF, JPG a PNG, PNG a JPG, redimensionar imágenes y comprimir PDF. Cada operación tiene un formulario con los campos específicos que necesita (ej: las dimensiones para resize, el checkbox "mantener proporción"). La versión final (009-ampliamos al maximo) es una web completa con estilos propios.

Las 8 operaciones reales de action.php

$operaciones = [
  'pdf_split'       => ['accept' => '.pdf',                    'icon' => '✂️'],
  'pdf_merge'       => ['accept' => '.pdf',   'multiple'=>true, 'icon' => '🔗'],
  'pdf_to_images'   => ['accept' => '.pdf',                    'icon' => '🖼️'],
  'images_to_pdf'   => ['accept' => '.jpg,.jpeg,.png',         'icon' => '📑'],
  'jpg_to_png'      => ['accept' => '.jpg,.jpeg',              'icon' => '🔄'],
  'png_to_jpg'      => ['accept' => '.png',                    'icon' => '🔄'],
  'images_resize'   => ['accept' => '.jpg,.png,.webp,.bmp',
    'fields' => [['name'=>'width','type'=>'number'],
                 ['name'=>'height','type'=>'number'],
                 ['name'=>'keep_ratio','type'=>'checkbox'],
                 ['name'=>'no_enlarge','type'=>'checkbox']]],
  'pdf_compress'    => ['accept' => '.pdf',                    'icon' => '📦'],
];

🛠 Tecnología real
