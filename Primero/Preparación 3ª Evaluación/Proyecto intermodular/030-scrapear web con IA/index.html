URL

02 ⬇

Fetch HTML

03 🧹

Limpiar

BeautifulSoup quita scripts, estilos y queda solo el texto útil.

04 🤖

Resumir IA

Inteligencia Artificial

Disciplina de la informática que estudia máquinas capaces de exhibir comportamiento inteligente.

  • Término acuñado por John McCarthy en 1956 en Dartmouth
  • Subdivide en IA débil (especializada) e IA fuerte (general)
  • Ramas: aprendizaje automático, NLP, visión artificial
  • Auge actual basado en redes neuronales profundas
  • Aplicaciones en salud, transporte, finanzas, asistentes

// stack técnico

  • Python 3 + Flask
  • requests + BeautifulSoup4
  • Ollama API · qwen2.5:7b
  • JSON estructurado de respuesta
  • Cache local de URLs ya scrapeadas

// el prompt

  • "Resume este artículo en 5 puntos"
  • "Devuelve un JSON con título y bullets"
  • Ventana 8K tokens · suficiente para Wikipedia
  • Streaming desactivado · respuesta completa
  • Truncado a 8000 caracteres si es muy largo
