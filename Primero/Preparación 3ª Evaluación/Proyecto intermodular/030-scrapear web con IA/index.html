🔍 Scrapear Web con IA

Python descarga HTML de cualquier URL y lo pasa a Ollama (qwen3.5:9b) para resumir la actividad de la empresa

📖 Qué hace realmente

Un script Python interactivo que pide una URL al usuario, descarga el HTML con requests.get y lo pasa íntegro a Ollama (qwen3.5:9b) con un prompt específico: "Resume en un párrafo cuál es la actividad de esa empresa. Solo el párrafo, sin comentar el código." La versión final lanza Ollama como subproceso (subprocess.run). Las versiones intermedias usan la API REST directamente con requests.post.

El código final real

import requests, subprocess

url   = input("Introduce una url: ")
html  = requests.get(url).text

prompt = (
    "Te voy a pasar el código fuente de una página web. "
    "Resume en un párrafo cual es la actividad de esa empresa u organización. "
    "Solo escribe el párrafo, no pongas nada de código, ni de HTML, "
    "ni comentes nada del código. Solo el párrafo en español.

"
    + html
)

result = subprocess.run(
    ["ollama", "run", "qwen3.5:9b", prompt],
    capture_output=True, text=True
)
print(result.stdout)

🛠 Librerías reales

📁 Progresión real

001-libreria requests.py ← GET básico 002-ollama web.md ← notas curl Ollama 003-peticion ollama.py ← API REST 004-todo junto.py ← requests + Ollama 005-afinar prompt.py ← prompt refinado 006-ejemplo interactivo.py ← versión final

⚠️ Nota técnica

El HTML completo se pasa como contexto al LLM — puede ser muy largo. Si la web tiene mucho JS/CSS, el modelo puede confundirse. El prompt explícitamente le pide que ignore el código y solo extraiga la actividad de la empresa.
