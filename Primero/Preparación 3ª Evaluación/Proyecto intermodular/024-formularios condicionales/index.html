PROYECTO 024 · LÓGICA RAMIFICADA

🌿 Formularios condicionales

El cuestionario cambia su flujo según las respuestas anteriores: si dices "no", saltas a otra pregunta. Cada camino lleva a su propio resultado.

// flujo ramificado de ejemplo

¿Has comprado online en el último mes?
SÍ
¿Lo recomendarías?
SÍ
+5 puntos · cliente fan
NO
+1 punto · neutro
NO
¿Por qué no?
PRECIO
cupón → reintentar
DUDA
FAQ + chat
// motor del salto condicional — logic.js
// Cada pregunta declara qué pregunta sigue
const flow = {
  q1: { text: "¿Compraste online?",
        next: { yes: "q2", no: "q3" } },
  q2: { text: "¿Lo recomendarías?",
        next: { yes: "END_FAN", no: "END_NEUTRO" } },
  q3: { text: "¿Por qué no?",
        next: { precio: "END_CUPON", duda: "END_FAQ" } }
};

function avanzar(qid, respuesta) {
  const destino = flow[qid].next[respuesta];
  if (destino.startsWith("END_")) return finalizar(destino);
  pintar(flow[destino]);
}

// claves del proyecto

  • JSON describe el árbol completo
  • Saltos por respuesta · sin reglas hardcoded
  • Cada nodo: pregunta o final
  • Resultados acumulan puntos por rama
  • Sin frameworks — JS vanilla
  • Backend PHP guarda el camino seguido
