💰 Panel de Control de Ventas

CRUD genérico PHP+SQLite con introspección de esquema vía PRAGMA — más asistente IA (qwen3.5:4b)

📖 Qué hace realmente

Un panel de gestión de ventas (clientes, productos, pedidos) con un giro técnico interesante: el CRUD es completamente genérico — no tiene código específico para ninguna tabla. Lee el esquema con PRAGMA table_info, detecta claves primarias y foráneas con PRAGMA foreign_key_list, y genera los formularios de alta/edición/borrado dinámicamente para cualquier tabla. Además incluye un asistente IA (ia.php) que proxifica preguntas a Ollama (qwen3.5:4b).

El CRUD genérico real

// El mismo panel funciona para cualquier tabla — sin cambiar código
