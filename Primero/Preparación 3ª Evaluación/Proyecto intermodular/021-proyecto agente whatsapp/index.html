// qué es

Una réplica HTML/CSS del UI de WhatsApp donde el lado «contacto» está respondido por un LLM local servido por Ollama. El usuario escribe, el agente responde como si fuera otra persona.

// stack

  • HTML + CSS · clon visual fiel
  • JS vanilla · sin frameworks
  • PHP proxy hacia Ollama local
  • Modelo: llama3.1:8b o similar
  • Historial en localStorage

// no es

  • NO integración con WhatsApp real
  • NO Twilio, NO WA Business API
  • NO envía mensajes a teléfonos
  • SOLO simulación visual web
💡 Idea: El proyecto sirve para practicar UI tipo chat y conectarlo con un modelo local. Es la base para luego integrar APIs de mensajería reales.
