PROYECTO 007 · PARSER DETERMINISTA

📄 Documentos a Bases de Datos

Parser Python por expresiones regulares para los PDFs del BOE con currículos de Formación Profesional. Sin IA, todo determinista. PDF → TXT → regex → modules.json → informe HTML.

REGEX006-proceso integral.py
# Patrones reales del parser RE_MODULOr"^\s*M[oó]dulo\s+Profesional\s*:\s*(.*)\s*$"
RE_ECTSr"(\d+)\s+(?:cr[eé]dito|ECTS)"
RE_CODIGOr"C[oó]digo\s*:\s*(\d{4})"
RE_RA_HEADERr"^\s*Resultados\s+de"
RE_RA_ITEMr"^\s*(\d+)\.\s*(.+?)\s*$"
RE_CRITERIOSr"^\s*Criterios\s+de"
RE_CRIT_ITEMr"^\s*([a-z])\)\s*(.+?)\s*$"
RE_DURACIONr"Duraci[oó]n\s*:\s*(\d+)"
RE_BULLETr"^\s*[-•·]\s*(.+)"
