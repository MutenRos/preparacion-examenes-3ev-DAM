🤖 Asistente de IA con Avatar 3D

Chatbot con avatar GLB en A-Frame, voz real (Web Speech API) y LLM local vía Ollama

📖 Qué hace realmente

Un chatbot web donde el personaje es un avatar 3D GLB animado en A-Frame 1.6.0. El usuario habla por el micrófono (SpeechRecognition), su mensaje va al backend PHP que lo envía a Ollama, y la respuesta se lee en voz alta (SpeechSynthesisUtterance) mientras el avatar anima sus morph targets sincronizado con el inicio/fin de la síntesis de voz. El proyecto tiene 24 iteraciones — de "load GLB básico" hasta el avatar animado final.

🎬 El truco del lip-sync

  • utterance.onstart → inicia loop de animación de morph targets
  • utterance.onend → detiene la animación
  • Los morph targets se alternan para simular habla
  • Debug overlay muestra el estado 3D en tiempo real

Fragmento de código real

// 024 - Morph targets animados SOLO durante la síntesis de voz
const utterance = new SpeechSynthesisUtterance(text);
utterance.onstart = () => {
    morphInterval = setInterval(() => {
        // anima morph targets del GLB avatar
        mesh.morphTargetInfluences[0] = Math.sin(Date.now() / 100) * 0.5 + 0.5;
    }, 50);
};
utterance.onend = () => {
    clearInterval(morphInterval);
    mesh.morphTargetInfluences[0] = 0;  // boca cerrada
};
speechSynthesis.speak(utterance);
