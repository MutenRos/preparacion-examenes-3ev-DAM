📝 Cuestionario Online de Inglés

Test de nivel CEFR (A1-C1) en PHP: preguntas desde Google Sheets, resultados en SQLite, panel admin

📖 Qué hace realmente

Un test de nivel de inglés online completo. Las preguntas se cargan en tiempo real desde un Google Sheets publicado como CSV. El usuario responde todas las preguntas en un único formulario; al enviar, PHP calcula la puntuación, la mapea al nivel CEFR correspondiente (A1 a C1), guarda el intento completo en SQLite (con detalle por pregunta) y muestra el resultado. Un admin.php lista todos los intentos.

La función de nivel real

function obtenerNivel(int $puntos): string {
    if ($puntos >= 1  && $puntos <= 10) return 'A1';
    if ($puntos >= 11 && $puntos <= 20) return 'A2';
    if ($puntos >= 21 && $puntos <= 30) return 'B1';
    if ($puntos >= 31 && $puntos <= 40) return 'B2';
    if ($puntos >= 41 && $puntos <= 50) return 'C1';
    return 'Sin nivel asignado';
}
// Rango: 50 preguntas, cada una vale 1 punto

🛠 Tecnología real

PHP + PDO SQLite Google Sheets CSV file_get_contents str_getcsv mb_internal_encoding

📊 Tablas SQLite

  • attempts — id, fecha, score, nivel
  • attempt_details — attempt_id, pregunta, respuesta_usuario, correcta, ok

🎯 Niveles CEFR

  • 1-10 puntos → A1
  • 11-20 → A2
  • 21-30 → B1
  • 31-40 → B2
  • 41-50 → C1
