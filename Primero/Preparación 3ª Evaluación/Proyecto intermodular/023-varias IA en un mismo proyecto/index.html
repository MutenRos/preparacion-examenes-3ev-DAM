🧩 Varias IA en un Proyecto

Libro de C++ generado por dos Ollama paralelos: llama3.1:8b para teoría, codellama:7b para código

📖 Qué hace realmente

No es un chatbot multi-modal. Es un generador automático de libro de texto de C++. Lee el temario de programación desde programacion.txt, y para cada sección llama a dos modelos distintos de Ollama: llama3.1:8b para el texto explicativo y codellama:7b para los ejercicios de código. El resultado se acumula en resultado.md. Un sistema de prompt de ~60 líneas impone normas pedagógicas estrictas: nunca anticipar conceptos futuros, cada paso debe compilar de forma autónoma, código incremental sobre el del paso anterior.

MODELO_TEXTO  = "llama3.1:8b"

# Reglas del sistema de escritura (extracto real):
# "Nunca anticipes conceptos futuros del temario."
# "Cada paso de código debe ser un programa COMPLETO y compilable."
# "Cada paso parte EXACTAMENTE del código del paso anterior."
# "El texto debe ser EN ESPAÑOL y de tono docente, cercano y claro."
# "Los ejercicios deben ir de menos a más dificultad dentro de cada tema."
