// pregunta del usuario
"Escribe una función que invierta una cadena en Python."
🦙
Llama 3.1
8B · generalista

Para invertir una cadena en Python lo más limpio es usar slicing. Es la forma idiomática y la más rápida en CPython:

def invertir(s):
    return s[::-1]

print(invertir("hola"))  # → "aloh"

El slice [::-1] recorre la cadena de derecha a izquierda con paso -1.

Tres formas equivalentes, ordenadas de más a menos pythónica:

def reverse_a(s): return s[::-1]
def reverse_b(s): return "".join(reversed(s))
def reverse_c(s):
    out = ""
    for ch in s: out = ch + out
    return out

# benchmark: a > b > c

Ojo: reverse_c es O(n²) por concat de strings.
