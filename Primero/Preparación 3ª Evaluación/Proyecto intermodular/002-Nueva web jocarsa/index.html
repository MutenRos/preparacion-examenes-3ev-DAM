El truco

Casi nadie implementa SMTP a mano hoy en día. Este proyecto lo hace para entender qué pasa por debajo cuando llamas a mail() o usas PHPMailer: el cliente de correo es solo una serie de comandos de texto sobre un socket TCP, con base64 para auth y TLS para cifrar.
