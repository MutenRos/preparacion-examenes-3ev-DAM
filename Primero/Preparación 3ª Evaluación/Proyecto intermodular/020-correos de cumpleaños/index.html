# Conectar con la base de datos y obtener cumples del día
hoy = datetime.now().strftime("%m-%d")
cur.execute("SELECT nombre, email FROM personas WHERE strftime('%m-%d', fecha_nac)=?", (hoy,))

for nombre, email in cur.fetchall():
    asunto = f"¡Feliz cumpleaños, {nombre}!"
    cuerpo = plantilla.format(nombre=nombre)

    with smtplib.SMTP_SSL("smtp.gmail.com", 465) as s:
        s.login(USER, APP_PASSWORD)
        s.send_message(build_message(email, asunto, cuerpo))

Tecnología

  • Python 3 + smtplib
  • SMTP_SSL · Gmail App Password
  • email.mime.multipart + HTML
  • SQLite con personas y fechas
  • Cron diario a las 09:00

Configuración

  • SMTP_USER · SMTP_PASS
  • Variables de entorno (no en código)
  • App Password en cuentas con 2FA
  • Plantilla HTML personalizable
