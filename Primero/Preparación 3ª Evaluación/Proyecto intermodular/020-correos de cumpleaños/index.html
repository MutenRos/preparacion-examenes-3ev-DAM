🎂 Correos de Cumpleaños

Python lee un Google Sheets CSV, detecta cumpleaños de hoy y envía email HTML personalizado por SMTP SSL

📖 Qué hace realmente

Script Python que cada día: (1) descarga un Google Sheets CSV con nombres, apellidos, emails y fechas de nacimiento, (2) compara solo día y mes con la fecha actual, (3) si hay coincidencia calcula la edad exacta y (4) envía un email HTML personalizado con smtplib.SMTP_SSL. El email tiene un diseño con gradiente, saludo personalizado con nombre + apellidos + edad y fallback en texto plano.

def is_birthday_today(birth_date: date, today: date) -> bool:
    return birth_date.day == today.day and birth_date.month == today.month

def years_old_today(birth_date: date, today: date) -> int:
    age = today.year - birth_date.year
    if (today.month, today.day) < (birth_date.month, birth_date.day):
        age -= 1
    return age
