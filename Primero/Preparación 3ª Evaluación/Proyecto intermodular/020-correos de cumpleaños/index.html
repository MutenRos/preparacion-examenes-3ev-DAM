Cake Correos de Cumpleanios

Email automatico de felicitacion por cumpleanos con Python

Descripcion

Lee agenda CSV, detecta cumpleanos del dia y envia email HTML personalizado via SMTP/Gmail.

Tecnologias

Python 3smtplibGmailCSVcron

Como ejecutar

pip install schedule python cumpleanos.py # Cron: 0 9 * * * python cumpleanos.py

Caracteristicas

  • Agenda CSV
  • Deteccion dia
  • Email HTML
  • Gmail SMTP
  • Ejecucion cron
  • Log envios
