🕸️ Scraper semántico

HTML → LLM con esquema JSON → datos estructurados sin selectores

1 · HTML crudo

<h1>iPhone 15 Pro</h1>
<span>1.299 €</span>
<p>Cámara de 48 Mpx,
chip A17 Pro</p>

2 · Prompt + schema

schema = {
  "producto": str,
  "precio_eur": float,
  "specs": list[str]
}

3 · JSON estructurado

{
  "producto":"iPhone 15 Pro",
  "precio_eur":1299.0,
  "specs":[
    "Cámara 48 Mpx",
    "Chip A17 Pro"
  ]
}

Sin escribir selectores CSS: el LLM entiende el contenido y lo encaja en pydantic.
