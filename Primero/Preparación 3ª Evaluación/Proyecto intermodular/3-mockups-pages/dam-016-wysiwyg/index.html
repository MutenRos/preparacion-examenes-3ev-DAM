¡Hola desde jocarsa WYSIWYG!

Este es un editor de texto enriquecido construido desde cero con JavaScript puro. Selecciona texto y usa los botones para formatearlo.

Prueba a poner en negrita, en cursiva, cambiar alineación o añadir listas. Todo funciona con document.execCommand() y un div[contentEditable].

cómo funciona

El núcleo del editor

  • Un div contenteditable="true" como área de texto
  • document.execCommand() para formato
  • Botones de toolbar con eventos onclick
  • El CSS formatea el resultado en tiempo real
  • Se integra en un <form> con <textarea> oculto
