PROYECTO 025 · LENGUAJE DE MARCADO PROPIO

📝 Multiformularios condicionales

Un editor escribe formularios en un lenguaje de marcado por indentación: cada tabulador es un nivel de anidación. PHP lo parsea y construye HTML real.

encuesta.dformindent = TAB
1# sintaxis: clave \t valor
2# anidación = nivel de tabuladores

// gramática del DSL

  • clave⇥valor separado por TAB
  • Indentación = anidación
  • campo · declara campo
  • tipo · text/radio/select…
  • opcion · cada choice
  • si.opcion → "X" · subform condicional
