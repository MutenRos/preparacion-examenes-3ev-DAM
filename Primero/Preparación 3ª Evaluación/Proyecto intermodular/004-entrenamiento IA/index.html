🧠 Entrenamiento de IA en CPU

Pipeline completo de fine-tuning LLM con LoRA sobre Qwen2.5-0.5B — funciona sin GPU (8 GB RAM)

📖 Qué hace realmente

Fine-tuning de Qwen/Qwen2.5-0.5B-Instruct usando LoRA (PEFT) con HuggingFace Transformers, entrenado íntegramente en CPU desactivando CUDA explícitamente. El dataset de entrenamiento es un fichero JSONL de preguntas y respuestas (002-base de conocimiento.jsonl). El pipeline tiene 4 fases: entrenar → fusionar adaptadores → inferencia → interfaz web Flask.

🛠 Librerías reales

Python 3 torch (CPU) transformers peft (LoRA) datasets flask

⚙️ Hiperparámetros reales

  • Modelo base: Qwen/Qwen2.5-0.5B-Instruct
  • Output dir: ./lora-cpu-lowram
  • Epochs: 15
  • Learning rate: 2e-4
  • Batch size: 1, Grad accum: 8
  • Max length: 256 tokens

python 004-fusion.py
