// arquitectura común

Una sola interfaz web. Detrás, Ollama sirve los modelos de texto y visión, mientras Stable Diffusion corre en su propio endpoint para la generación. PHP enruta cada petición al backend correcto según la pestaña activa, y devuelve JSON con texto o base64 de imagen.

// stack

  • PHP frontend con tabs
  • Ollama serving texto + visión
  • Stable Diffusion en GPU/CPU
  • Imágenes vía base64 → img.src
  • Carga de archivos para visión

// modelos disponibles

  • llama3.1:8b · texto general
  • llava:7b · imagen → texto
  • qwen2.5:7b · texto rápido
  • SD-1.5 · texto → imagen
  • Todos locales, sin API externa
