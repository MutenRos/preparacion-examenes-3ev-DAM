Aunque no lo usaremos mucho en programacion, y se supone que ya lo sabemos, es importante conocer los componentes basicos de un ordenador, y como se conectan entre si. En este caso, vamos a ver la estructura de una placa base Raspberry Pi, que es un ordenador, reducido a su minima expresion, y que es muy util para proyectos de modelado e impresion 3D, pero un ordenador al fin y al cabo.

Usaremos el ultimo modelo que adquirí, una raspberry pi 5 con 16gb de ram, que es la que uso para mis proyectos de IA. pues cuenta con un HAILO8, que es un coprocesador de IA que me permite hacer cosas muy interesantes. Pero eso es otra historia, es un componente añadido, no es parte de la estructura basica de un ordenador.

En la placa base, tenemos varios componentes importantes:
- CPU: El cerebro del ordenador, donde se procesan las instrucciones. En la Raspberry Pi 5, es un procesador ARM Cortex-A76.
- RAM: La memoria de acceso aleatorio, donde se almacenan los datos temporales que el CPU necesita para funcionar. En este caso, tenemos 16GB de RAM LPDDR4X.
- Buses de datos: Son las autopistas que conectan todos los componentes entre si, permitiendo que se comuniquen y transfieran datos. En la Raspberry Pi, tenemos varios buses, como el bus USB, el bus HDMI, y el bus GPIO.
- Puertos de expansión: Permiten conectar dispositivos externos, como teclados, ratones, discos duros, etc. En la Raspberry Pi, tenemos puertos USB, un puerto HDMI, y un puerto Ethernet.  
- Almacenamiento: Aunque la Raspberry Pi no tiene un disco duro tradicional, utiliza una tarjeta microSD para almacenar el sistema operativo y los datos.

Para tener un PC funcional, solo nos faltaria conectarle:
- Una fuente de alimentacion, que en este caso es un adaptador USB-C que proporciona 5V y 3A.
- Un monitor, que se conecta a traves del puerto HDMI.
- Un teclado y un raton, que se conectan a traves de los puertos USB.
- Una tarjeta microSD con el sistema operativo instalado.

Conociendo estos componentes y su interconexion, podemos entender mejor como funciona un ordenador, y como podemos utilizarlo en nuestros proyectos de modelado e impresion 3D. Por ejemplo, podemos diseñar carcasas personalizadas para la Raspberry Pi, o crear accesorios que se conecten a los puertos GPIO para interactuar con otros dispositivos.
Ademas, entender la estructura de un ordenador nos ayuda a optimizar el rendimiento de nuestros proyectos, ya que podemos elegir componentes adecuados y configurar el sistema operativo de manera eficiente.
