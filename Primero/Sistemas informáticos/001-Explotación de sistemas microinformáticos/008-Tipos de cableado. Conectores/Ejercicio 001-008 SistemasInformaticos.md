Contexto
En nuestra clase de explotación de sistemas microinformáticos, hemos estudiado los tipos de cableado y conectores utilizados en las redes informáticas. Como aficionados a jugar con Raspberrys y modelado 3D, sabemos que la comprensión de estos elementos es crucial para configurar y mantener nuestras instalaciones tecnológicas. En este ejercicio práctico, exploraremos cómo los diferentes tipos de cableados y conectores se utilizan en sistemas microinformáticos.

Enunciado paso a paso
Identificación de conectores comunes:

RJ45: Este es el conector más común para cables Ethernet. Es el tipo típico de cable utilizado en redes locales.
RJ11: Aunque parecido al RJ45, este es el conector usado para los teléfonos domésticos.
Tipos de cableados:

Ethernet 8P8C: Este tipo de cableado utiliza un conector RJ45 y se utiliza comúnmente en redes locales.
USB A-B: Existen varias versiones de este cableado (1.0, 1.1, 2.0, 3.0), todos ellos azules y utilizados para conectar dispositivos USB entre sí.
Otros conectores:

Minijack: Este es el conector utilizado en tarjetas de sonido profesionales para los puertos de audio.
e-SATA: Este tipo de puerto se utiliza para transferir datos a alta velocidad.
HDMI y Displayport: Estos conectores gráficos son utilizados para conectar dispositivos como televisores o monitores.

Ademas de los puertos virtuales que hemos visto en la unidad, los ordenadores cuentan con una serie de puertos fisicos, que son los que permiten conectar dispositivos externos al ordenador. Estos puertos fisicos estan conectados a la placa base mediante cables y conectores, que son los que vamos a ver en este ejercicio.
Es importante aprender a identificar tanto los puertos fisicos(conectores) como los tipos de cableado que se utilizan para conectar los dispositivos al ordenador, ya que esto nos permitira configurar y mantener nuestras instalaciones tecnologicas de manera eficiente.
Empezaremos por los conectores mas comunes:
- RJ45: Este es el conector mas comun para cables Ethernet. Es el tipo tipico de cable utilizado en redes locales. Tiene 8 pines y se utiliza para conectar dispositivos a una red local.
- RJ11: Aunque parecido al RJ45, este es el conector usado para los telefonos domesticos. Tiene 4 pines y se utiliza para conectar el telefono a la linea telefonica.
Luego tenemos los tipos de cableado mas comunes:      
- Ethernet 8P8C: Este tipo de cableado utiliza un conector RJ45 y se utiliza comunmente en redes locales. Tiene 8 pines y se utiliza para conectar dispositivos a una red local.
- USB A-B: Existen varias versiones de este cableado (1.0, 1.1, 2.0, 3.0), todos ellos azules y utilizados para conectar dispositivos USB entre si. El conector A es el que se conecta al ordenador, y el conector B es el que se conecta al dispositivo.
Ademas de los conectores y cableados mas comunes, existen otros conectores que se utilizan en sistemas microinformaticos:
- Minijack: Este es el conector utilizado en tarjetas de sonido profesionales para los puertos de audio. Tiene 3 pines y se utiliza para conectar auriculares, altavoces, etc.
- e-SATA: Este tipo de puerto se utiliza para transferir datos a alta velocidad. Tiene 7 pines y se utiliza para conectar discos duros externos.
- HDMI y Displayport: Estos conectores graficos son utilizados para conectar dispositivos como televisores o monitores. El conector HDMI tiene 19 pines y el conector Displayport tiene 20 pines.
En resumen, conocer los diferentes tipos de conectores y cableados es fundamental para la explotacion de sistemas microinformaticos, ya que nos permite configurar y mantener nuestras instalaciones tecnologicas de manera eficiente. Ademas, este conocimiento es especialmente relevante para nuestro hobby de jugar con Raspberrys y modelado 3D, ya que nos permite conectar y utilizar diferentes dispositivos en nuestros proyectos.
